## 准备工作

### 环境准备

安装以下内容

-   [git](https://git-scm.com/)
-   [node](https://nodejs.org/en/)

### 克隆仓库

直接克隆本仓库或者 fork 后，安装项目依赖

```shell
git clone https://github.com/ultimatepasserby/assaultlily-sheepgame.git
cd assaultlily-sheepgame
npm install -g yarn
yarn install
```
安装完成后调试运行，复制链接到浏览器即可

```shell
yarn dev:diy
```
